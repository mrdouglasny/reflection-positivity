**Date**: 2026-07-20. **Purpose**: pin the generic lattice reflection-positivity theorem in

the **keystone for pphi2 cylinder OS3** (`hRP`): once it lands, the pphi2 asym-lattice RP

(`IsReflectionPositive μ θ mPos := ∀ F, Measurable[mPos] F → Integrable(F·F∘θ) → 0 ≤ ∫ F·(F∘θ) dμ`,
`Abstract.lean:126`).

## The measure class

Finite site type `Λ` (`[Fintype Λ]`), configurations `Ω := Λ → ℝ`, a **site reflection**
`r : Λ ≃ Λ` that is an involution and edge-preserving, lifting to `θ : Ω → Ω`, `θ φ := φ ∘ r`.
Measure
  `μ ∝ exp(-½ Σ_{⟨x,y⟩∈E} J_{xy} (φ_x − φ_y)² − Σ_x V(φ_x)) ∏_x dφ_x`
with `J_{xy} ≥ 0` (ferromagnetic), `V : ℝ → ℝ` bounded below (for integrability; **no parity
constraint on V** — this is Simon's generality, and it is exactly what pphi2 needs since its
`V = a²:P:` is not the kinetic term). Half-space σ-algebra `mPos` = generated by `{φ_x : x ∈ Λ₊}`
where `Λ₊` is the positive-time half; the reflection plane sits **between sites** (the "`Nt` even"
condition — required so `r` has no fixed sites on the plane).

## The two proof routes (choose one — see §Vet)

**Route A — Gaussian crossing perfect square (classical GJ 6.2.2).**
1. **Action split (easy).** `V` is single-site, `r` permutes sites ⟹ `Σ V(φ_x)` is θ-invariant;
   the kinetic bonds split into (within-Λ₊) + (within-Λ₋ = reflected Λ₊) + **crossing bonds**
   (each joins `x∈∂Λ₊` to `r x∈∂Λ₋`). [pphi2 `action_decomposition` is the V-side, generalizes.]
2. **Half integration.** For `mPos`-measurable `F`, `∫ F·(F∘θ) dμ` factors over the plane config
   into `∫_{plane} |G(φ₊,plane)|²-shaped · (crossing kernel)` — the within-halves + V pieces give
   the modulus-square (identical `Λ₊`/`Λ₋` structure via θ-symmetry), leaving the crossing kernel.
3. **HARD CORE — the crossing perfect square.** The crossing term
   `exp(−½ Σ_{x∈∂Λ₊} J_x (φ_x − φ_{r x})²) = exp(−½ J_x φ_x²)·exp(−½ J_x φ_{rx}²)·exp(Σ J_x φ_x φ_{rx})`.
   The quadratic self-terms absorb into the halves; the **cross term** `exp(Σ J_x φ_x φ_{rx})`
   with `J_x ≥ 0` is a **positive-definite kernel** in `(φ_∂, φ_{r∂})`, so the whole integral is a
   perfect square `⟨g, g⟩ ≥ 0`. This is the generic form of pphi2's current N-specific axiom
   `gaussian_rp_cov_perfect_square` (`OSProofs/OS3_RP_Lattice.lean`, still an axiom on the square
   side too). ⚠ **Provable-or-axiom decision (the crux):** `exp(J φ ψ)`, `J ≥ 0`, is PD by
   Hubbard–Stratonovich: `exp(J φ ψ) = 𝔼_{λ~N(0,J)}[exp(λφ)exp(λψ)]` (or the finite-dim
   `exp(½⟨v,Av⟩)` with `A` the crossing coupling PSD via `J_x ≥ 0`). Mathlib has Gaussian-integral
   machinery; whether this closes cleanly or stays a **vetted axiom** (as on the square side) is
   the main open question. If it stays an axiom, the WIN is still large: ONE generic PD/perfect-
   square axiom replaces per-geometry ones, and everything else (split, integration, assembly) is
   generic and proved.

**Route B — Osterwalder–Seiler transfer positivity.**
RP ⟺ the transfer operator `T` across the reflection plane is positive self-adjoint, via
Feynman–Kac `∫ F·(F∘θ) dμ = ⟨T^{1/2}Φ_F, T^{1/2}Φ_F⟩ ≥ 0`. pphi2 already builds the asym transfer
system (`asymTransferSystem`), proved the transfer operator **positive kernel + self-adjoint +
compact** (`AsymL2Operator`, `AsymJentzsch`), and has the trace dictionary
(`TransferSystem.twoPoint_dictionary`, `GroundBridge`). Route B would reuse this heavily. Cost:
connecting the **full RP form over arbitrary positive-time `F`** (not just two-point functions)
to the transfer operator needs a Feynman–Kac representation of `∫ F·(F∘θ) dμ` as an operator
inner product — the trace dictionary currently covers two-point observables, so this needs a
generalization (`F` = product of slice observables ⟹ `Φ_F` = the corresponding vector). Route B
avoids Hubbard–Stratonovich entirely; its hard core is instead the FK representation lemma.

## VET OUTCOME (Gemini 3.1-pro, 2026-07-20) — Route A, and the hard core is AXIOM-FREE

**Route A chosen.** Route B is an "illusion of reuse": defining the Feynman–Kac vector `Φ_F`
for *arbitrary* positive-half `F` forces the same μ₊/μ₋ split + crossing isolation as Route A,
just with added functional-analytic overhead. Route A stays in measure theory (Lean-friendly).

**Deliverable 3 is PROVABLE — drop the axiom.** Use **per-edge Hubbard–Stratonovich**, NOT
Taylor series, NOT a PD-matrix axiom, NOT multivariate Gaussian identities:
- The crossing sum factors over boundary edges:
  `exp(Σ_{x∈∂Λ₊} J_x φ_x φ_{rx}) = ∏_x exp(J_x φ_x φ_{rx})`.
- 1D HS per edge (`J ≥ 0`):
  `exp(J φ ψ) = (2π)^{-1/2} ∫_ℝ exp(√J z φ)·exp(√J z ψ)·exp(−z²/2) dz`
  (this is just the 1D Gaussian MGF `∫ exp(√J z (φ+ψ)) dN₀(z) = exp(J(φ+ψ)²/2)` split via
  `(φ+ψ)² = φ² + ψ² + 2φψ`, the `φ²/ψ²` absorbing into the halves — a clean elementary identity).
- Introduces an auxiliary standard Gaussian `dN₀(z)` on `ℝ^{|∂Λ₊|}`; by Fubini the full integral
  becomes `∫ dN₀(z) · [∫ F(φ₊) e^{Σ√J z_x φ_x} dμ₊] · [∫ (θF)(φ₋) e^{Σ√J z_x φ_{rx}} dμ₋]`.
- By θ-symmetry (μ₊ ≅ μ₋ under `r`) the second bracket = the first, so the integrand is
  `|I(z)|² ≥ 0`, hence `∫ dN₀(z) |I(z)|² ≥ 0`. **No axioms; only Fubini + the 1D Gaussian MGF.**

**Decomposition confirmed flawless:** (a) NN interaction ⟹ the halves strictly factorize modulo
the single crossing-bond layer (the self-terms `e^{−½J φ²}` belong to their own half); (b) the
"plane between sites / reflect across bonds" (Nt even) condition is exactly what yields the clean
`e^{J φ_x φ_{rx}}` coupling — reflecting *through* sites would instead give a `δ(φ_x − φ_{rx})`
coupling needing a different representation, which this setup avoids; (c) `Σ V(φ_x)` is
θ-invariant with **no parity condition on V** (r permutes site labels) — confirming Simon's
generality (parity is only for the internal Z₂ symmetry, not spatial RP).
